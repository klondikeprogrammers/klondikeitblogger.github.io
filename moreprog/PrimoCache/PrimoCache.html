PrimoCache

Описание:

PrimoCache — программа для увеличения производительности работы жестких дисков, твердотельных
SSD-накопителей и USB-флешек путем перенаправления запросов на чтение и запись данных с
аппаратного кеш-модуля устройств на оперативную память компьютера.
PrimoCache - дополнительная схема кэширования программного обеспечения, которая взаимодействует с
системной памятью для обеспечения кэширования данных для дисков. Это улучшает производительность
системы за счетпрозрачного хранения данных в памяти, так что будущие запросы на эти данные можно
обслуживать быстрее. PrimoCache кэширует данные на основе логического блока (смещения на диске), в
то время как диспетчер кэширования Windows кэшируется на основе виртуального блока (смещения
внутри файла). PrimoCache также может использовать небезопасную память ОС, которая теряется в
32-разрядной Windows при использовании 4 ГБ памяти или более. Принцип Одним из основных
компонентов PrimoCacheявляется драйвер фильтра класса хранения, который находится в стеке
хранилища, перехватывая запросы ввода-вывода для данных на диске.

Операционные системы:

  • Windows 7, 8.1, 10 (32/64-bit)
